8 Existe-t-il plusieurs types de tableaux en PHP, si oui lesquels ?

Oui, en PHP, il existe plusieurs types de tableaux :

  • Tableaux indexés : Ce sont les tableaux les plus couramment utilisés. Les éléments sont accessibles par leur position (indice) numérique.

    $fruits = array("pomme", "banane", "orange");

    echo $fruits[0]; // Affiche "pomme"

  • Tableaux associatifs : Dans ce type de tableau, les éléments sont associés à des clés plutôt qu'à des indices numériques. Chaque élément est constitué d'une paire clé-valeur.

    $personne = array(
    "nom" => "Doe",
    "prénom" => "John",
    "âge" => 25);

    echo $personne["nom"]; // Affiche "Doe"

  • Tableaux multidimensionnels : Ce sont des tableaux qui contiennent d'autres tableaux à l'intérieur. Ils permettent de représenter des structures de données complexes.

    $matrice = array( array(1, 2, 3), array(4, 5, 6), array(7, 8, 9) );

    echo $matrice[1][2]; // Affiche 6

13 Quelle est la différence entre les instructions « require » et « include » en PHP

"Require" est plus strict et arrête l'exécution en cas d'erreur, tandis que "include" est plus permissif et continue l'exécution en affichant un avertissement en cas d'erreur. Le choix entre les deux dépend de la nécessité et de l'importance du fichier inclus dans le fonctionnement du script.
